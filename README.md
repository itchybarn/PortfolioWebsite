# Portfolio Website

#### Created by Jack Wagenheim

This is a portfolio website that I created to keep all my work in one place. It can be found at https://www.jackwagenheim.com or https://www.itchybarn.com, with the main domain being itchybarn.com.

## Frontend

Built with React and TypeScriptl.
It is being managed by Vite.

Any time Vite sses a request with a URL beginning with "/api/", it will know to forward that back to Flask.

## Backend

A Flask app is served by Gunicorn

## Deployment

Run on AWS
